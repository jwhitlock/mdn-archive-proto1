<h2 id="核心工具">核心工具</h2>

<div class="column-container">
<div class="column-half">
<h3 id="Page_Inspector(页面检查器)">Page Inspector(页面检查器)</h3>

<p><a href="/en-US/docs/Tools/Page_Inspector"><img alt="The all-new Inspector panel in Firefox 57." src="https://mdn.mozillademos.org/files/15481/57-inspector.png" style="display:block; height:243px; margin-left:auto; margin-right:auto; width:425px" /></a></p>

<p>查看和编辑页面内容和布局。可视化页面的许多方面，包括盒子模型，动画和网格布局。</p>
</div>

<div class="column-half">
<h3 id="Web_Console">Web Console</h3>

<p><a href="/en-US/docs/Tools/Web_Console"><img alt="The all-new Console in Firefox 57." src="https://mdn.mozillademos.org/files/15483/57-console.png" style="display:block; height:239px; margin-left:auto; margin-right:auto; width:425px" /></a></p>

<p>查看由页面和JavaScript交互记录的信息</p>
</div>
</div>

<div class="column-container">
<div class="column-half">
<h3 id="JavaScript_Debugger">JavaScript Debugger</h3>

<p><a href="/en-US/docs/Tools/Debugger"><img alt="The all-new Firefox 57 Debugger.html" src="https://mdn.mozillademos.org/files/15485/57-debugger.png" style="display:block; height:239px; margin-left:auto; margin-right:auto; width:425px" /></a></p>

<p>停止，浏览，检查和修改页面中运行的JavaScript。</p>
</div>

<div class="column-half">
<h3 id="Network_Monitor(网络监视器)">Network Monitor(网络监视器)</h3>

<p><a href="/en-US/docs/Tools/Network_Monitor"><img alt="The Network panel in Firefox 57 DevTools." src="https://mdn.mozillademos.org/files/15487/57-network.png" style="display:block; height:239px; margin-left:auto; margin-right:auto; width:425px" /></a></p>

<p>查看加载页面时的网络请求。</p>
</div>
</div>

<div class="column-container">
<div class="column-half">
<h3 id="Performance_Tools(性能工具)">Performance Tools(性能工具)</h3>

<p><a href="/en-US/docs/Tools/Performance"><img alt="" src="https://mdn.mozillademos.org/files/14536/performance.png" style="display:block; height:1026px; margin-left:auto; margin-right:auto; width:425px" /></a></p>

<p>分析网站的响应，JavaScript程序和布局性能。</p>
</div>

<div class="column-half">
<h3 id="Responsive_Design_Mode(响应式设计)">Responsive Design Mode(响应式设计)</h3>

<p><a href="/en-US/docs/Tools/Responsive_Design_Mode"><img alt="Responsive Design mode in Firefox 57." src="https://mdn.mozillademos.org/files/15491/57-rdm.png" style="display:block; height:266px; margin-left:auto; margin-right:auto; width:425px" /></a></p>

<p>查看网站或app在不同终端和网络类型上的外观和行为。</p>
</div>

<h2 id="更多工具">更多工具</h2>

<p>这些开发者工具也内置在Firefox中。 不像上面的“核心工具”一样常用。</p>

<div class="twocolumns">
<dl>
 <dt><a href="/en-US/docs/Tools/Memory">Memory(内存)</a></dt>
 <dd>判断哪些对象正在使用内存。</dd>
 <dt><a href="/en-US/docs/Tools/Storage_Inspector">Storage Inspector(存储检查器)</a></dt>
 <dd>检查存在于页面中的cookie，本地存储，indexedDB和session。</dd>
 <dt><a href="/en-US/docs/Tools/DOM_Property_Viewer">DOM Property Viewer(DOM属性查看器)</a></dt>
 <dd>检查页面上 DOM 属性，函数等.</dd>
 <dt><a href="/en-US/docs/Tools/GCLI">Developer Toolbar(开发者工具栏)</a></dt>
 <dd>开发者工具的命令行界面。</dd>
 <dt><a href="/en-US/docs/Tools/Eyedropper">Eyedropper(页面取色器)</a></dt>
 <dd>从页面中选择一种颜色。</dd>
 <dt><a href="/en-US/docs/Tools/Scratchpad">Scratchpad(便笺)</a></dt>
 <dd>Firefox内置的文本编辑器，可让您编写和执行JavaScript。</dd>
 <dt><a href="/en-US/docs/Tools/Style_Editor">Style Editor(样式编辑器)</a></dt>
 <dd>查看和编辑当前页面的CSS样式。</dd>
 <dt><a href="/en-US/docs/Tools/Shader_Editor">Shader Editor</a></dt>
 <dd>查看和编辑<a href="/en-US/docs/Web/WebGL">WebGL</a>使用的顶点和片段着色器。</dd>
 <dt><a href="/en-US/docs/Tools/Web_Audio_Editor">Web Audio Editor</a></dt>
 <dd>检查音频上下文中音频节点的图形，并修改其参数。</dd>
 <dt><a href="/en-US/docs/Tools/Screenshot_tool">Taking screenshots(截图)</a></dt>
 <dd>对整个页面或单个元素的屏幕截图。</dd>
</dl>
</div>

<hr />
<h2 id="连接开发者工具">连接开发者工具</h2>

<p>如果您使用<a href="/en-US/docs/Tools/Keyboard_shortcuts#Opening_and_closing_tools">快捷键</a>或菜单打开开发者工具, 他们将定位由当前活动选项卡托管的文档。但是，你也可以将这些工具连接到其他目标上，无论是在当前的浏览器中，还是在不同的浏览器中，甚至是不同的设备上。</p>

<div class="twocolumns">
<dl>
 <dt><a href="/en-US/docs/Tools/about:debugging">关于：调试</a></dt>
 <dd>在浏览器中运行附加组件。</dd>
 <dt><a href="/en-US/docs/Tools/Remote_Debugging/Debugging_Firefox_for_Android_over_Wifi">连接到Android的Firefox</a></dt>
 <dd>将开发者工具连接到在Android设备上运行的Firefox。</dd>
 <dt><a href="/en-US/docs/Tools/Working_with_iframes">连接到iframe</a></dt>
 <dd>将开发者工具连接到当前页面中的特定iframe。</dd>
 <dt><a href="/en-US/docs/Tools/Valence">连接到其他浏览器</a></dt>
 <dd>将开发者工具连接到Android上的Chrome和iOS上的Safari。</dd>
</dl>
</div>

<hr />
<h2 id="在浏览器调试">在浏览器调试</h2>

<p>默认情况下，开发者工具将附加到网页或Web应用程序。 但是，你也可以将它们连接到浏览器。 这对于浏览器和附加组件开发很有用。</p>

<div class="twocolumns">
<dl>
 <dt><a href="/en-US/docs/Tools/Browser_Console">浏览器控制台</a></dt>
 <dd>查看由浏览器本身和插件记录的信息，并在浏览器里运行JavaScript代码。</dd>
 <dt><a href="/en-US/docs/Tools/Browser_Toolbox">浏览器工具箱</a></dt>
 <dd>将开发工具附加到浏览器本身。</dd>
</dl>
</div>

<hr />
<h2 id="扩展devtools">扩展devtools</h2>

<p>开发者工具被设计为可扩展的。Firefox附加组件可以访问开发者工具，扩展现有工具和添加新工具。借助远程调试协议，你可以实现自己的调试工具，来调试网站，或使用Firefox工具调试不同的网站。</p>

<div class="twocolumns">
<dl>
 <dt><a href="/en-US/docs/Tools/Example_add-ons">devtools附加组件示例</a></dt>
 <dd>使用这些示例来了解如何实现devtools附加组件。</dd>
 <dt><a href="/en-US/docs/Tools/Adding_a_panel_to_the_toolbox">devtools中添加新的面板</a></dt>
 <dd>编写一个组件，添加一个新的面板到工具箱。</dd>
 <dt><a href="https://wiki.mozilla.org/Remote_Debugging_Protocol">远程调试协议</a></dt>
 <dd>用于将Firefox开发者工具连接到调试目标的协议，例如Firefox或Firefox OS设备的实例。</dd>
 <dt><a href="/en-US/docs/Tools/Editor">源码编辑器</a></dt>
 <dd>Firefox中内置的代码编辑器，可嵌入你的附加组件中。</dd>
 <dt><a href="/en-US/docs/Tools/Debugger-API"><code>Debugger</code> 接口</a></dt>
 <dd>一个JavaScript代码观察其他JavaScript代码的执行的API。 Firefox开发者工具使用这个API来实现JavaScript调试器。</dd>
 <dt><a href="/en-US/docs/Tools/Web_Console/Custom_output">Web Console自定义输出</a></dt>
 <dd>如何扩展和自定义 <a href="/en-US/docs/Tools/Web_Console">Web Console</a> 和 <a href="/en-US/docs/Tools/Browser_Console">Browser Console</a>的输出。</dd>
</dl>
</div>
