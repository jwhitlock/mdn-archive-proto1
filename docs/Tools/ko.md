 <div class="summary">
  <h3 id="What's_new_in_Aurora.3F" name="What's_new_in_Aurora.3F">파이어 폭스 개발자 에디션의 새로운 기능은 무엇입니까?</h3>
  <p><a href="/en-US/Firefox/Developer_Edition">파이어폭스 개발자 에디션</a>은&nbsp;개발자들을 위한 파이어폭스의 맞춤형 버전이며, 최신 버전 파이어폭스의 모든 기능과, 실험용 개발자 도구를 포함합니다. 현재 파이어폭스 개발자 에디션에서 제공되는 기능은 다음과 같습니다.</p>
  <ul>
   <li>
    <p><a href="/en-US/docs/Tools/Page_Inspector#.3A.3Abefore_and_.3A.3Aafter">페이지 인스펙터에서 ::before와 ::after 같은 가상 요소 보기 </a></p>
   </li>
   <li>
    <p><a href="/en-US/docs/Tools/Style_Editor#Source_map_support">이제 기본으로 CSS 소스 맵을 사용 할 수 있습니다.</a></p>
   </li>
   <li>
    <p><a href="/en-US/docs/Tools/Page_Inspector#Element_popup_menu_2">페이지 인스펙터에서 "DOM 속성 보기"추가</a></p>
   </li>
  </ul>

 <div class="column-half">
  <h2 id="Creating" name="Creating">만들기</h2>
  <p>웹사이트나 웹 애플리케이션을 위한 저작 도구입니다.</p>
  <dl>
   <dt>
    <a href="/en-US/docs/Tools/Scratchpad">스크래치패드 | Scratchpad</a></dt>
   <dd>
    자바스크립트를 작성하고 실행해볼 수 있는 파이어폭스에 내장 편집기입니다.</dd>
   <dt>
    <a href="/en-US/docs/Tools/Style_Editor">스타일 편집기 | Style Editor</a></dt>
   <dd>
    현재 페이지의 CSS 스타일을 보거나 편집합니다.</dd>
   <dt>
    <a href="/en-US/docs/Tools/Shader_Editor">셰이더 편집기 | Shader Editor</a></dt>
   <dd>
    <a href="/en-US/docs/Web/WebGL">WebGL</a>이 사용하는 버텍스나 부분 셰이더를 열어보고 편집합니다.</dd>
   <dt>
    <a href="/en-US/docs/Tools/Web_Audio_Editor">웹 오디오 편집기 | Web Audio Editor</a></dt>
   <dd>
    오디오 컨텍스트에 있는 오디오 노드 그래프를 검사하거나 매개변수를 수정합니다.</dd>
  </dl>
 </div>
 <div class="column-half">
  <h2 id="Exploring" name="Exploring">탐색 및 디버깅</h2>
  <p>웹사이트나 웹 애플리케이션을 검사하고, 탐색하거나 디버깅 합니다.</p>
  <dl>
   <dt>
    <a href="/en-US/docs/Tools/Web_Console">웹 콘솔 | Web Console</a></dt>
   <dd>
    웹 페이지의 로그 메세지를 확인하거나, 자바스크립트를 사용하여 페이지와 상호 작용합니다.</dd>
   <dt>
    <a href="/en-US/docs/Tools/Page_Inspector">페이지 인스펙터 | Page Inspector</a></dt>
   <dd>
    페이지의 HTML과 CSS를 보거나 수정할 수 있습니다.</dd>
   <dt>
    <a href="/en-US/docs/Tools/Debugger">자바스크립트 디버거 | JavaScript Debugger</a></dt>
   <dd>
    페이지의 자바스크립트 실행을 멈추거나, 단계별로 검사 혹은 수정할 수 있습니다.</dd>
   <dt>
    <a href="/en-US/docs/Tools/Network_Monitor">네트워크 모니터 | Network Monitor</a></dt>
   <dd>
    페이지를 로드 할 때 어떤 네트워크 요청이 일어나는지 확인하세요.</dd>
   <dt>
    <a href="/en-US/docs/Tools/Storage_Inspector">스토리지 인스펙터 | Storage Inspector</a></dt>
   <dd>
    페이지에 존재하는 쿠키, 로컬 저장소, indexedDB 및 세션 저장소를 검사합니다.</dd>
   <dt>
    <a href="/en-US/docs/Tools/GCLI">개발자 도구 | Developer Toolbar</a></dt>
   <dd>
    개발자 도구를 위한 커맨드라인 인터페이스입니다.</dd>
   <dt>
    <a href="/en-US/docs/Tools/3D_View">3D 뷰 | 3D View</a></dt>
   <dd>
    <span class="short_text" id="result_box" lang="ko"><span class="hps atn">페이지를 3차원으로 시각화 하여 보여줍니다.</span></span></dd>
   <dt>
    <a href="/en-US/docs/Tools/Eyedropper">아이드롭 | Eyedropper</a></dt>
   <dd>
    페이지에서 색상을 선택하세요.</dd>
   <dt>
    <a href="/en-US/docs/Tools/Working_with_iframes">iframe 작업</a></dt>
   <dd>
    컨텐츠가 어떤 iframe에서 표시될지 결정하세요.</dd>
  </dl>

   <dt>
    <a href="/en-US/docs/Tools/WebIDE">웹 통합 개발 환경 | WebIDE</a></dt>
   <dd>
    App Manager를 Firefox 33 이후로 계속 사용할 수 있도록 한 대체품입니다.</dd>
   <dt>
    <a href="/en-US/docs/Tools/Firefox_OS_Simulator">파이어폭스OS 시뮬레이터 | Firefox OS Simulator</a></dt>
   <dd>
    파이어폭스OS가 설치된 디바이스 없이도 데스크탑에서 Firefox OS용 애플리케이션을 실행하고 디버그 할 수 있습니다.</dd>

   <dt>
    <a href="/en-US/docs/Tools/Remote_Debugging/Firefox_for_Android">안드로이드용 파이어폭스에서 디버깅</a></dt>
   <dd>
    안드로이드용 파이어폭스와 연결하여 디버깅 할 수 있는 개발 도구입니다.</dd>
   <dt>
    <a href="/en-US/docs/Tools/Valence">발란스 | Valence</a></dt>
   <dd>
    iOS의 사파리나 안드로이드의 크롬과 연결하여 디버깅 할 수 있는 개발 도구입니다.</dd>
  </dl>

   <dd>

 <div class="column-half">
  <h2 id="Debugging_the_browser" name="Debugging_the_browser">브라우저 디버깅하기</h2>
  <p>보통의 경우, 개발자 도구는 웹 페이지나 웹 애플리케이션을 디버깅 하지만 브라우저에 연결하여 브라우저나 애드온 프로그램 개발에 유용하게 사용할 수도 있습니다.</p>
  <dl>
   <dt>
    <a href="/en-US/docs/Tools/Browser_Console">브라우저 콘솔 | Browser Console</a></dt>
   <dd>
    브라우저나 애드온 프로그램에서 발생한 로그 메시지를 확인하세요. 페이지에서 직접 자바스크립트 코드를 실행할 수도 있습니다.</dd>
   <dt>
    <a href="/en-US/docs/Tools/Browser_Toolbox">브라우저 툴박스 | Browser Toolbox</a></dt>
   <dd>
    브라우저에 개발자 도구를 추가하세요.</dd>
  </dl>
 </div>
 <div class="column-half">
  <h2 id="Extending_the_devtools" name="Extending_the_devtools">개발자 도구 확장하기</h2>
  <p>개발자 도구는 확장이 용이하도록 디자인 되었습니다. 파이어폭스 애드온은 기존의 도구를 확장하거나 새로운 도구를 추가하기 위한 컴포넌트 및 개발자 도구를 다룰 수 있으며, 원격 디버깅 프로토콜을 통해 여러분만의 도구를 사용해 웹사이트를 디버그 하거나, 혹은 파이어폭스에서 다른 디바이스를 디버그 하는 여러분만의 클라이언트나 서버를 구현할 수도 있습니다.</p>
  <dl>
   <dt>
    <a href="https://wiki.mozilla.org/Remote_Debugging_Protocol">원격 디버깅 프로토콜 | Remote Debugging Protocol</a></dt>
   <dd>
    원격 디버깅 프로토콜은 파이어폭스 개발자 도구가 파이어폭스나 파이어폭스OS 장치로 연결할 수 있게 해줍니다.</dd>
   <dt>
    <a href="/en-US/docs/Tools/Editor">코드 편집기 | Source Editor</a></dt>
   <dd>
    여러분이 개발한 애드온 프로그램에 파이어폭스 코드 편집기를 내장할 수 있습니다.</dd>
   <dt>
    <a href="/en-US/docs/Tools/Debugger-API"><code>디버그&nbsp;</code>인터페이스 | The Debugger Interface</a></dt>
   <dd>
    작성한 자바스크립트 코드의 실행 결과를 관찰할 수 있는 API입니다. 파이어폭스 개발자 도구는 이 API를 사용해서 자바스크립트 디버거를 만들었습니다.</dd>
   <dt>
    <a href="/en-US/docs/Tools/Web_Console/Custom_output">웹 콘솔 아웃풋 커스터마이징</a></dt>
   <dd>
    <a href="/en-US/docs/Tools/Browser_Console">브라우저 콘솔</a>과 <a href="/en-US/docs/Tools/Web_Console">웹 콘솔</a>을 커스터마이징 하는 방법을 알아보세요.</dd>
  </dl>

<h2 id="More_resources" name="More_resources">그 외의 리소스</h2>
<p>이 섹션의 리소스들은 모질라 개발자 도구 팀에서 제공하는 것이 아니지만 웹 개발자들에게 잘 알려진 것들이며, 이 중에는 파이어폭스 애드온 프로그램에 포함된 것도 있습니다. 전체 목록을 보려면 <a href="https://addons.mozilla.org/en-US/firefox/extensions/web-development/">addons.mozilla.org 사이트의 "Web Developer" 카테고리</a>를 확인하세요.</p>
<div class="column-container">
 <div class="column-half">
  <dl>
   <dt>
    <a href="https://www.getfirebug.com/">Firebug</a></dt>
   <dd>
    매우 잘 알려진 강력한 웹 개발 도구이며, 자바스크립트 디버거 및 HTML과 CSS뷰어, 그리고 네트워크 모니터를 포함합니다.</dd>
   <dt>
    <a href="/en-US/docs/DOM_Inspector">DOM Inspector</a></dt>
   <dd>
    웹 페이지의 DOM이나 XUL 윈도우를 탐색하고 편집하며, 점검해 볼 수 있는 도구입니다.</dd>
   <dt>
    <a href="https://addons.mozilla.org/en-US/firefox/addon/web-developer/">Web Developer</a></dt>
   <dd>
    다양한 웹 개발 도구를 브라우저의 메뉴나 툴바에 추가하세요.</dd>
  </dl>
 </div>
 <div class="column-half">
  <dl>
   <dt>
    <a href="https://webmaker.org/en-US/tools/">Webmaker Tools</a></dt>
   <dd>
    웹 개발을 시작하는 사람들을 위한 도구 모음입니다.&nbsp;모질라에서 개발된 도구들로 이루어져 있습니다.</dd>
   <dt>
    <a href="http://www.w3.org/Status.html">W3C Validators</a></dt>
   <dd>
    W3C 웹사이트는 여러분의 <a href="http://validator.w3.org/">HTML</a>과 <a href="http://jigsaw.w3.org/css-validator/">CSS</a>를 포함한&nbsp;웹사이트 정합성을 확인하는 여러 도구들을 제공하고 있습니다.</dd>

    <a href="http://www.jshint.com/">JSHint</a></dt>

  </dl>
